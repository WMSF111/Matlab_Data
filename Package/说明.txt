Package 文件说明（Data processing 项目）

说明原则
- 本说明覆盖 Package 目录下当前全部文件。
- 每个文件给出一句用途说明，便于快速定位。
- Modeling/PLS 下大量函数来自 libPLS 工具链，优先按“建模核心、验证、变量选择、可视化、工具函数、示例数据”分组说明。

一、根目录
1. 说明.txt
- Package 总体说明文档（本文件）。

二、Black white（黑白矫正）
1. Black white/Black_White_Processing.m
- 读取 NIR csv，按分组求平均并进行黑白矫正，输出 post_processing_data.xlsx。

三、Feature_select（特征筛选）
1. Feature_select/feature_select_corr_topk.m
- 按每个波段与目标值 Pearson 相关系数绝对值排序，选择 Top-K 特征。

四、Pre_processing（光谱预处理）
1. Pre_processing/SG.m
- Savitzky-Golay 平滑/导数预处理。
2. Pre_processing/MSC.m
- 多元散射校正（MSC）。
3. Pre_processing/SNV.m
- 标准正态变量变换（SNV）。
4. Pre_processing/normalization.m
- 数据归一化处理。
5. Pre_processing/Standardization.m
- 标准化（通常是零均值单位方差）。
6. Pre_processing/mean_centering.m
- 均值中心化。
7. Pre_processing/detrending.m
- 去趋势处理。
8. Pre_processing/move_smooth.m
- 滑动平滑。

五、Sample_selsect（样本选择）
1. Sample_selsect/KS.m
- Kennard-Stone 样本划分（训练/测试集选择）。
2. Sample_selsect/SPXY.m
- SPXY 样本划分接口文件（当前文件为空占位）。

六、Wave_select（波段选择）
1. Wave_select/SPA.m
- SPA（Successive Projections Algorithm）变量/波段筛选。
2. Wave_select/PCA.m
- PCA 降维/主成分提取。

七、Modeling/PLS（libPLS 核心工具）
A. 建模核心
1. Modeling/PLS/pls.m
- PLS 回归主函数。
2. Modeling/PLS/plsnipals.m
- NIPALS 方式求解 PLS。
3. Modeling/PLS/plsval.m
- 用训练好的 PLS 模型进行预测。
4. Modeling/PLS/plslda.m
- PLS-DA 分类模型。
5. Modeling/PLS/opls.m
- OPLS 建模。
6. Modeling/PLS/mwpls.m
- 移动窗口 PLS。
7. Modeling/PLS/ecr.m
- ECR（弹性成分回归）建模。
8. Modeling/PLS/ecrpred.m
- ECR 模型预测。
9. Modeling/PLS/oscwold.m
- OSC（Wold 版本）正交信号校正。
10. Modeling/PLS/oscfearn.m
- OSC（Fearn 版本）正交信号校正。
